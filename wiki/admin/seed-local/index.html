Local Seed Tool

Creates local, system-conform trailer reference posts for validation without pushing live changes.

Seed Tool Disabled

This local migration tool is intentionally disabled by default to protect the working baseline. To enable it on purpose, open this page with ?enableSeedTool=1.

Seed: Trailer Reference Pack

This updates the local trailer-reference seed set authored by the lead admin account. You can run a single biome example or the full multi-category trailer knowledge pack.

  • Uses renamed BoundLore image assets under /public/images/wiki-seed/
  • Writes BLMETA fields for update phase, patch tag, trailer source URL, and subcategory
  • Full pack targets biomes, creatures, and items with system-compatible wiki posts
